# AGENTS

## Overview
The agent system provides a modular framework for building autonomous agents that can perform complex tasks autonomously. Each agent is defined by a set of configuration files and can be orchestrated across multiple runtimes. This document serves as the foundational reference for authors, developers, and users interacting with the agent ecosystem.

## Purpose
The primary goals of the agent framework are to enable rapid prototyping of AI-driven workflows, provide extensibility through custom skills, ensure robust fault tolerance, and support seamless integration with existing toolchains. By abstracting the core orchestration logic, developers can focus on defining high-level agent behaviors rather than low-level infrastructure details.
