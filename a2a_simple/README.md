# A2A Simple Example
This module demonstrates the invocation of dummy greeting agent via A2A protocol using a2a client.

## Steps:

- Setup agent
- Wrap it in agent executor
- Configure agent skills, card and capabilities
- Wrap it in startlette app to run as a2a server (JSON-RPC) via starlette
- Use the test_client file to invoke the a2a server which in turn calls the agent
